如果你遇到了任何 bug 或者对功能有疑问，请 在 GitHub 上反馈给开发者


提交反馈之前请先确认你的浏览器和 pakku 已经是最新版本。

如果你安装了其他扩展程序（比如 TamperMonkey、哔哩哔哩助手、某些代理软件等），请尝试禁用它们，看看问题是否解决。

请在 pakku 的选项页面里选择“重置所有设置”，并重启浏览器，看看问题是否解决。


下面的调试信息包括当前系统环境和你现在的设置。如果需要，请在 bug report 中附上它们。

如果你发现播放视频时 pakku 右上角的图标出现红色提示，请点击它，并在 bug report 中附上相关报错信息。
